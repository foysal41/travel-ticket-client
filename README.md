# TravelTicket 

A full-stack ticket booking platform where users can search and book travel tickets, vendors can manage and publish tickets, and admins can approve or reject ticket listings.

## Features

### Authentication & Authorization

* Better Auth authentication system
* User registration and login
* Role-based access control
* User, Vendor, and Admin dashboards
* Protected routes

### User Features

* Browse approved tickets
* Search and filter tickets
* View ticket details
* Departure countdown timer
* Book tickets with quantity selection
* Stripe payment integration
* View booked tickets
* Track booking status

### Vendor Features

* Add new tickets
* Upload ticket images using ImgBB
* Manage added tickets
* Edit ticket information
* View requested bookings
* Accept or reject booking requests
* Vendor profile page

### Admin Features

* Manage all tickets
* Approve tickets
* Reject tickets
* Update ticket verification status
* View all vendor-submitted tickets

### Public Features

* Responsive homepage
* Hero section
* Popular routes section
* Advertisement section
* Latest tickets section
* Responsive navigation
* Responsive footer
